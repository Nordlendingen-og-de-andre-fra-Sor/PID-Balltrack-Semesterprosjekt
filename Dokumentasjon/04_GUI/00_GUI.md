#  Graphical User Interface (GUI)
Et GUI (Graphical User Interface) er et grafisk brukergrensesnitt som gjør det mulig for operatører å overvåke og styre et system på en intuitiv måte ved hjelp av visuelle elementer som knapper, menyer og statusindikatorer.

GUI-et viser normalt: 
[1] Setpunkt (SP) 
[2] Målt posisjon (PV)
[3] Status for start/stopp
[4] Regulatorens utsignal/pådrag (u)
[5] Type regulatorform (parallell, serie, pi, pd eller kun p). I tillegg kan Kp, Ki og Kd justeres i sanntid 
[6] Slik at man raskt kan se effekten på respons, oversving og innstillingstid. 

# Hvorfor er GUI viktig?
•	Enklere betjening
•	Færre operatørfeil
•	Raskere feilsøking
•	Bedre oversikt og sikkerhet
